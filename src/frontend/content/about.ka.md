# წიგნის მიმოხილვა - ჩვენ შესახებ

კეთილი იყოს თქვენი მობრძანება წიგნის მიმოხილვაზე — პლატფორმა წიგნების მოყვარულებისთვის.

## ჩვენი მისია

ჩვენ გვწამს, რომ ყოველი წიგნი იმსახურებს გააზრებულ მიმოხილვას. ჩვენი პლატფორმა მკითხველებს აძლევს საშუალებას გააზიარონ თავიანთი შეხედულებები, საყვარელი ციტატები და რეკომენდაციები.

## ფუნქციები

- შექმენით და გამოაქვეყნეთ წიგნის მიმოხილვები Markdown ფორმატით
- გამოყეთ თქვენი საყვარელი ციტატები წაკითხული წიგნებიდან
- ატვირთეთ გარეკანის სურათები მიმოხილვების ვიზუალურად გასალამაზებლად
- სრული ორენოვანი მხარდაჭერა ქართულ და ინგლისურ ენებზე

## ტექნოლოგიები

წიგნის მიმოხილვა აგებულია თანამედროვე ტექნოლოგიებით, მათ შორის Next.js, .NET და PostgreSQL.
